# Prueba técnica backend - Similar Products

API en Spring Boot que devuelve el detalle de los productos similares a uno dado.

## Requisitos

- Java 21
- Maven (o usar el wrapper `mvnw` incluido)

## Arrancar

```bash
./mvnw spring-boot:run
```

La app corre en el puerto 5000. Las APIs existentes del enunciado se esperan en `http://localhost:3001` (se puede cambiar en `application.yml`).
